eaaomk 2021-04-25 18:00:00 审计速成财务会计

# 财务会计与审计

财务会计与审计专业毕业生的就业前景十分广阔，在各行各业财政、金融、税收、经贸等企事业单位从事会计、审计、财务管理等工作。如：保险业统计员、审计师、银行检测员、收益管理员、预算控制分析员、注册公共会计师、金融主管、教授、公共会计师、证券经纪人、系统分析员、税务专家、财政官员、信托会计师、保险业者、政府部门、银行/金融部门、工业、六大会计公司、咨询公司、非盈利性组织、政府部门、索赔协调员、客户信贷员、成本会计师、信贷与回收专家、政府会计师等。

但目前的行业形势下，财务会计与审计的门槛均已提高，类似于移动软件开发，门槛也已提高，对于低端程序员或者说是初级从业者不太友好。

但工资未来可期，外部审计工作可能会加班，比较累，内部审计较为良好，财务会计良好。

# 初学者入门、外行转审计

如果没有这方面的经验，可以选择考一个高含金量的证书，如果要做审计，还需要会计方面的知识，注册会计师证是必不可少的了。 考一个注会证书不仅仅在专业知识能力上能够提升不少，同时对自己以后的工作、就业等都非常有优势。

# 注册会计师考试内容

注册会计师考试分为专业阶段和综合阶段，专业阶段考试内容包括六科：会计、审计、财管、税法、经济法、战略；综合阶段综合能力测试试卷一、试卷二。

# 注册会计师CPA考试介绍

注册会计师专业阶段需要在5年内通过6门科目，考生可以选择其中任意科目进行组合报考，也可一次报考六科。但是由于注会考试有一定难度，一般考生会选择2~3年考完。注册会计师考试一般在4月份报名，10月份安排考试，由于2021年考试时间改革，提前至8月份了。由于题主是在职人员，建议可以选择其中的部分科目报考，例如可以根据自己的职业需要来选择，可优先报考会计+审计。如果觉得时间还比较充裕，可以再加一门难度较小的税法或经济法。关于注册会计师考试科目搭配方案，还可参考以下：

# CPA 与ACCA

# 定义

ACCA，全称特许公认会计师公会，在国内被考生称为国际注册会计师。

CPA，全称注册会计师，是专业技能型证书。

# 含金量

CPA是专业技能型证书，在中国从事审计业务且唯一拥有签字权的证书。

ACCA是综合性证书（可以做学术），从事国际审计业务，可以转为某个国家的注册会计师，ACCA在中国做审计业务没有签字权。

# 薪资待遇

ACCA，没有资历，年薪20-30万；

ACCA，资历丰富，年薪50万往上，高的甚至好几百万。

CPA，没有资历，没有签字权，年薪10-20万；

CPA，资历丰富，有签字权，年薪50万往上。

# 发展前景

ACCA可以在会计师事务所、企业、金融业任职，甚至可以去做学术，就业范围非常广。如果去外企，ACCA是非常好的敲门砖。而且ACCA的国际性受到“走出去”的大型国企或民企的青睐。

CPA必须在会计事务所从事审计工作，同时，CPA拥有唯一签字权在国内很受欢迎。

# 适合人群

# CPA

1.考证学霸：无论是什么证书，对学霸而言没有难度大小之分，都一样！比如上海立信会计金融学院教CPA老师，同时还考出像ACCA，CFA，CGA，FRM，CFA等财务证书。

2.会计师事务所人员：对于会计事务所的人来说，CPA是通往职场巅峰的必要条件，每当想放弃时，就会想到完全没有在底层拼搏的苦。

3.有目标的大学生或研究生： 大三大四学CPA，或者研一开始学CPA，都是有目标的学生。这些目标明确的学生，早就筹备好了，CPA只是其中一环，多少艰难困苦都要把CPA考出来不可。

4.有职场梦想的人 当自己有了想成为一个注册会计师的职业梦想后，就需要付出多倍的努力，唯有这样才能最终收获成功的硕果。

# ACCA

1.英国留学生、会计硕士：ACCA是英国的财会考试，如果你正好在英国留学，并且就读于会计相关专业，那考起试来比如会事半功倍。

2.财务主管：大环境所致，财经培训市场日益红火，高级管理人员更受青睐，因此，ACCA必然是首选之一。

3.银行工作人员：银行属于金融业，和ACCA的考试内容有着千丝万缕的联系。并且，对于常跳槽的银行职工来说，手里有价值高的证书，无疑是晋升的法宝。

4.CPA持证人：CPA和ACCA侧重的内容不同，更是两种不同的财务思维，两证在手，可以说能够在财务这行横着走了。
